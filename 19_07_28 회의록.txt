1. 스마트 현관문 ( 스마트 도어락 + 스마트 미러 )
요약 : 밖에서 여는 문제 + 안에서 나갈 때의 상황도 고려한 아이디어.
내용 : 
	1. 밖 : 여는 방법을 핸드폰(어플)로 열거나, 캠을 이용해서 사람 인식 후 열기.
	2. 안 : 움직임 감지 센서를 통해서 현재 밖의 날씨, 내가 이용한 교통수단의 정보를 알려주기.
	         미세먼지가 많은 날에는 미세먼지 마스크가 나오는 수납장.
	         비오는 날에는 우산이 나오는 수납장.
	         챙겨야 할 게 있으면 따로 마련한 수납장 등등..

2. 스마트 신호등 ( 교통 흐름 최적화 )
요약 : 신호등 체계를 유동적으로 해서 자동차와 사람의 흐름을 편하게 해줌.
내용 : 
	카메라로 차와 사람을 인식하여, 신호등의 시간을 조정함.
	ex) 많은 차가 있으면 시간을 더 배분하고, 적은 차가 있으면 시간을 줄인다.
기타 : 
	제일 많은 공감으로 인해 유력한 아이디어.
	하지만 지자체(연수구청)에 문의해서 실현가능성과 테스트가능성을 체크 해봐야함.

3. 스마트 그늘막
요약 : 그늘막을 사람이 관리하는 것이 아니라, 자동으로 관리하는 시스템.
내용 :
	1. 비가오는 경우 : 필터를 통과한 빗물 모으기, 비, 바람으로 인한 피해로 인해 그늘막을 접는다.
	2. 해가 떠 있는 경우 : 태양광 발전으로 전기를 생산, 생산한 전기로 그늘막을 펼친다.
	3. 온도가 높거나 건조한 경우 : 물을 미세입자로 만들어 안개를 생성하여 기화를 통해 주변 온도를 낮춤.

6. 빔 신호등
요약 : 빔을 이용해 횡단보도를 그림.
내용 :
	1. 초록불(사람이 횡단할 때)에 빔으로 횡단보도를 그림.
	2. 빨간불(사람이 기다릴 때)에 횡단보도를 지움.

	비가 오는 날에 빔이 보일 지 걱정.

8. AR 드론
요약 : 드론을 조종할 때 AR을 이용함.
내용 : 
	조종기를 통해 드론을 조종하지만 드론에 카메라가 달려 있어서 조종기에 그 카메라 화면이 나타남.
	그 화면에 가상 물체나 이미지를 나타내어 게임이나 다른 응용분야에 활용.
기타 :
	구현을 해도 사용 분야가 게임분야밖에 떠오르지 않음.
